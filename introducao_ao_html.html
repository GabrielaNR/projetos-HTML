O que é o HTML?

A Linguagem de Marcação de Hipertexto (HTML) é uma linguagem de computador que compõe a maior parte das páginas da internet e dos aplicativos online. Um hipertexto é um texto usado para fazer referência a outros textos, enquanto uma linguagem de marcação é composta por uma série de marcações que dizem para os servidores da web qual é o estilo e a estrutura de um documento.
O HTML não é considerado uma linguagem de programação, já que ele não pode criar funcionalidades dinâmicas. Ao invés disso, com o HTML, os usuários podem criar e estruturar seções, parágrafos e links usando elementos, tags e atributos.

Estrutura básica do HTML

Documentos HTML são arquivos que terminam com uma extensão .html ou .htm. Um navegador lê o arquivo HTML e renderiza o seu conteúdo para que os usuários da internet possam vê-lo.
Todas as páginas HTML possuem uma série de elementos, que consistem num conjunto de tags e atributos. Os elementos HTML são os tijolos de construção de uma página da internet. Uma tag diz para o navegador onde um elemento começa e termina, enquanto um atributo descreve as características de um elemento.
As três principais partes de um elemento são:

  • Tag de abertura – usada para dizer onde um elemento começa a ter efeito. A tag é cercada de colchetes angulares para abertura e fechamento. Por exemplo, use a tag de abertura <p> para criar um parágrafo.
  • Conteúdo – essa é a parte que os usuários verão.
  • Tag de fechamento – igual à tag de abertura, mas com uma barra antes do nome do elemento. Por exemplo, </p> para encerrar um parágrafo.

Exemplos de Tags

Tags são elementos utilizados para marcar e estruturar o conteúdo da página web. Elas são representadas por meio de pares de sinais de menor "<" e maior ">" que envolvem o conteúdo que iremos marcar.
Exemplos:

  • <i>Deixa o texto em itálico.</i>
  • <strong>Deixa o texto em negrito.<strong>
  • <img src="link da imagem" alt="texto alternativo">: Exibir uma imagem - <img src="endereço_da_imagem.jpg" alt="Texto alternativo.">
  • <hr />: Insere uma linha entre os conteúdos.
  • <h1></h1>: São os cabeçalhos do texto, vão de h1 à h6, sendo h1 o maior e h6 o menor em tamanho.
  • <p>< /p>: Demarcam o parágrafo de um texto.
  • <blockquote>< /blockquote>: Usado para fazer citações em um texto.

Atributos das Tags

Atributos são utilizados nas tags para identificá-las e agrupá-las, tornando mais fácil sua edição e melhorando o site semânticamente. Exemplos:

  • id: Identifica um elemento, como seu "nome".
  • class: Identifica um conjunto de elementos agrupados em uma classe.
  • src: Especifica o caminho para a origem de um recurso, como uma imagem ou um script.
  • href: Contém o destino de um link, seja para página interna ou externa.
  • alt: Fornece um texto alternativo para uma imagem, que é exibido caso ela não seja carregada.

Listas Ordenadas e Não Ordenadas

Listas Ordenadas são listas enumeradas em ordem. Exemplo:

  1. Eu sou o primeiro item da lista.
  2. Eu sou o segundo item da lista.

As listas Não Ordenadas não são enumeradas:

  • Eu sou um item da lista.
  • Eu sou outro item da lista.

Links

Os links são endereços para outras páginas web, imagens ou tópicos dentro de uma mesma página.
Um link básico é criado envolvendo o texto (ou outro conteúdo) que você quer transformar em um link dentro de um elemento <a>, e dando-lhe um atributo href, (também conhecido como Hypertext Reference, ou target) que conterá o endereço da Web para o qual você deseja que o link aponte.
Atributos:

  • <a href=" "> : referencia links externos.
  • <a href="#secao1"> : referencia links de dentro da mesma página (ou seções da página).
  • Target: o uso do "target" define se o link vai ser aberto na mesma aba do site de origem ou em uma diferente.
